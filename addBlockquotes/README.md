# Usage
- macOS 用户就双击打开， 把 `txt` 拖进小窗口里回车，就会在你拖进去的 `txt` 的当前目录下生成处理好后的

- windows 用户除了双击打开使用，还可以直接把 `txt` 拖到 `exe` 的图标上使用，效果是一样的，但是更方便
